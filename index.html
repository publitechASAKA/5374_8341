▼5374.jpについて
【埼玉県朝霞市版 5374.jp】
今日は（明日は）何のゴミの日？
このゴミの正しい捨て方は？　これらの疑問を、アプリ上で解消しましょう。
引っ越してきたばかりの方、日本語がむずかしい方のゴミ出しをサポートします。

賛助協力：朝霞市役所 資源リサイクル課

5374.jpの製作&提供：CodeForKanazawa

朝霞市版製作：PublitechASAKA（パブリテックあさか）

使い方

お住まいの地域を選択することで、ごみ収集日の予定が更新されます。

色でごみのジャンルを表示

一番近いごみの日と資源名を上から順に表示しています。

ごみの資源名をタップすると、捨てられるごみの代表的な品名と、注意点を見ることができます

ごみの捨て方がわからないとき

「▼このごみ、どう出す？」に参考となるリンクをまとめました。

提供されるゴミ情報について

朝霞市資源リサイクル課にて公開しているデータをもとに、アプリに実装しました。

ライセンスについて

Code for Kanazawaについて

コードで、世界をHappyに。
各地域には様々な課題があり、解決するためのコミュニティ（団体）も数多くあります。
課題の中には、ITやデザインの力で解決できるものも多くありますが、全てのコミュニティにそのスキル（技術）が備わっているわけではありません。

Code for Kanazawa（CfK）は、市民の課題を集め、その課題を整理・分析した上で、メンバーが実際に課題解決となるソフトウェアやハードウェア（仕組みや方法）を開発します。
私たちは、デザインをして、ソフトウェアコードを書き、課題を解決するサービスを完成させるのです。さらに、そのサービスを提供し続ける力も持ちます。

CfKは市民のための組織です。行政や民間企業の影響を受けずに中立・公益の立場から物事を判断するよう心がけています。

公式サイト
http://www.codeforkanazawa.org/

PublitechASAKA（パブリテックあさか）について

IT技術を活用して、朝霞市での”くらし”が住み良くなることを目指す、有志の市民活動団体です。
市民目線の課題意識を元に、行政や、他の市民活動団体と連携しながら、課題解決を試みます。
5374.jp朝霞市版のアプリが、市民の皆様に活用され、ごみ出しマナーの改善に役立つことを願います。
